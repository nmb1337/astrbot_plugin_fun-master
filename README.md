# astrbot_plugin_fun

一个 AstrBot 积分玩法插件，提供随机聊天奖励、签到、查询、排行、管理员增减分和兑换提醒。

## 功能

- 聊天随机奖励：按概率触发积分奖励（默认 5%）。
- 奖励积分范围：随机发放在设定区间内（默认 1-10）。
- 连续签到加成：连签越久奖励越高（支持配置每日加成与上限）。
- 基础积分管理：签到、查询、排行。
- 抽奖玩法：消耗积分抽奖，支持一/二/三等奖积分和概率配置。
- 抽奖保底：连续 N 次未中奖后，下次必出三等奖（N 可配置）。
- 管理员管理：管理员可加分、扣分、设置概率、设置奖励范围。
- 积分兑换审核：兑换单状态支持已申请、已处理、已完成，并支持管理员确认。

## 指令

指令组：`/积分`（别名：`/point`、`/points`、`/jifen`）

- `/积分 签到`
- `/积分 查询 [QQ]`
- `/积分 排行 [人数]`
- `/积分 抽奖 [次数]`
- `/积分 兑换 [提醒内容]`
- `/积分 兑换状态 [兑换单号]`
- `/积分 设置` 查看当前概率、奖励范围、冷却和签到奖励

管理员指令：

- `/积分 加分 <QQ> <积分>`
- `/积分 扣分 <QQ> <积分>`
- `/积分 设置概率 <0-100>`
- `/积分 设置范围 <最小值> <最大值>`
- `/积分 设置奖项 <等级1-3> <奖项积分> <概率%>`
- `/积分 设置抽奖消耗 <积分>`
- `/积分 设置保底 <N>`
- `/积分 设置兑换积分 <积分>`
- `/积分 设置兑换通知 <QQ>`
- `/积分 兑换处理 <兑换单号> [备注]`
- `/积分 兑换完成 <兑换单号> [备注]`
- `/积分 兑换待处理 [数量]`

## 配置

插件支持 WebUI 可视化配置，配置文件为 [_conf_schema.json](_conf_schema.json)。

默认配置：

- 随机奖励概率：5%
- 随机奖励区间：1-10
- 聊天奖励冷却：30 秒
- 签到基础奖励：8
- 连签每日加成：2（加成上限 30）
- 抽奖消耗：10
- 一等奖：100 积分，概率 1%
- 二等奖：30 积分，概率 5%
- 三等奖：10 积分，概率 20%
- 抽奖保底阈值：10（0 表示关闭）
- 固定兑换积分：100
- 排行榜默认人数：10

## 存储

插件使用 AstrBot KV 存储保存积分数据（用户积分、签到日期、随机奖励设置等）。
