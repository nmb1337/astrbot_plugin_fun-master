# astrbot_plugin_fun

一个 AstrBot 积分玩法插件，提供随机聊天奖励、签到、查询、排行、管理员增减分和兑换审核提醒。

下载插件并启用后，会启动本地积分看板网站，默认地址：

- http://127.0.0.1:16666

网页中可直接查看：

- 每个人的积分（昵称、QQ、积分、连签）
- 兑换申请与处理记录（已申请、已处理、已完成）

## 功能

- 聊天随机奖励：按概率触发积分奖励（默认 5%），群聊需 @ 机器人 才会触发，私聊默认触发。
- 奖励积分范围：随机发放在设定区间内（默认 1-10）。
- 连续签到加成：连签越久奖励越高（支持配置每日加成与上限）。
- 基础积分管理：签到、查询、排行。
- 抽奖玩法：消耗积分抽奖，支持一/二/三等奖积分和概率配置。
- 抽奖保底：连续 N 次未中奖后，下次必出三等奖（N 可配置）。
- 管理员管理：管理员可加分、扣分、设置概率、设置奖励范围。
- 积分兑换审核：兑换单状态支持已申请、已处理、已完成，并支持管理员确认。

## 指令

注意：命令已改为直连写法，不再使用 `/积分 xxx` 前缀。

普通指令：

- `/签到`
- `/查询 [QQ]`
- `/排行 [人数]`
- `/抽奖 [次数]`
- `/兑换 [提醒内容]`
- `/兑换状态 [兑换单号]`
- `/设置` 查看当前概率、奖励范围、冷却和签到奖励

管理员指令：

- `/加分 <QQ> <积分>`
- `/扣分 <QQ> <积分>`
- `/群控状态`
- `/白名单加 [群号,群号2,...]`
- `/白名单删 [群号,群号2,...]`
- `/白名单列表`
- `/黑名单加 [群号,群号2,...]`
- `/黑名单删 [群号,群号2,...]`
- `/黑名单列表`
- `/设置概率 <0-100>`
- `/设置范围 <最小值> <最大值>`
- `/设置奖项 <等级1-3> <奖项积分> <概率%>`
- `/设置抽奖消耗 <积分>`
- `/设置保底 <N>`
- `/设置兑换积分 <积分>`
- `/设置兑换通知 <QQ>`
- `/兑换处理 <兑换单号> [备注]`
- `/兑换完成 <兑换单号> [备注]`
- `/兑换待处理 [数量]`
- `/看板地址`
- `/重启看板`

## 配置

插件支持 WebUI 可视化配置，配置文件为 [_conf_schema.json](_conf_schema.json)。
建议先在 WebUI 中设置 `redeem_notify_qq` 为有效 QQ 号（支持输入纯数字或包含 QQ 号的文本，插件会自动提取）。

### 群白名单/黑名单

- `group_whitelist` 非空时：仅白名单群会启用积分功能。
- `group_blacklist` 优先级更高：命中黑名单的群始终禁用。
- `group_enable_private` 控制私聊是否启用积分功能。
- 白名单/黑名单命令支持逗号一次传多个群号，也支持在群聊里不带参数直接对当前群生效。

你可以通过管理员指令快速维护名单，也可以在 WebUI 配置里直接编辑。

### 本地网站看板

插件会同时启动一个本地网站看板，默认端口来自 `dashboard_port`（默认 16666）。

- 地址：`http://127.0.0.1:16666`
- 可通过 `/看板地址` 获取当前实际地址
- 如需限制访问，可设置 `dashboard_token`，访问时带 `?token=你的口令`

注意：部分 Chromium 浏览器会拦截 6666 端口（ERR_UNSAFE_PORT）。默认已经使用 `16666`，如有冲突可再改成其他端口。

默认配置：

- 随机奖励概率：5%
- 随机奖励区间：1-10
- 聊天奖励冷却：30 秒
- 签到基础奖励：8
- 连签每日加成：2（加成上限 30）
- 抽奖消耗：10
- 一等奖：100 积分，概率 1%
- 二等奖：30 积分，概率 5%
- 三等奖：10 积分，概率 20%
- 抽奖保底阈值：10（0 表示关闭）
- 固定兑换积分：100
- 排行榜默认人数：10

## 存储

插件使用 AstrBot KV 存储保存积分数据（用户积分、签到日期、随机奖励设置等）。
